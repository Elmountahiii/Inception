Welcome to My Dockerized Website

A simple website running in a Docker container

About This Site

This is a demonstration of a simple static website that has been containerized with Docker. The site includes basic HTML and CSS styling to create a clean, modern look.

Why Docker?

Docker makes it easy to package applications and their dependencies into containers, ensuring that they run the same way regardless of the environment. This makes deployment simple and consistent.

Getting Started

To run this website, you just need to build the Docker image and run it. The Dockerfile takes care of setting everything up for you!
